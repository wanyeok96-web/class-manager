학급·학생 기록을 다루는 프로그램입니다.

---

## 먼저 정하기: 둘 중 어떤 걸 원하세요?

| | 설명 |
|---|------|
| **① 내 컴퓨터에서만 쓰기** | 인터넷 주소 필요 없음. 제일 간단합니다. |
| **② 인터넷 링크로 쓰기** | 다른 사람도 브라우저 주소로 접속하게 할 수 있습니다. 준비가 조금 필요합니다. |

---

## ① 내 컴퓨터에서만 쓰기 (가장 쉬움)

1. 이 폴더 안에 있는 **`run_local.bat`** 파일을 **더블클릭**합니다.
2. 검은 창이 잠깐 뜨고, 곧 **브라우저(크롬 등)** 가 열리면 성공입니다.
3. 주소창에 `localhost:8501` 같은 주소가 보이면, 그 화면에서 프로그램을 쓰시면 됩니다.

**안 될 때**

- `Python을 찾을 수 없습니다` 비슷한 말이 나오면 → [python.org](https://www.python.org/downloads/) 에서 Python을 설치한 뒤, 다시 `run_local.bat` 을 실행해 보세요. 설치할 때 **“Add Python to PATH”** 에 체크하는 것이 좋습니다.
- 방화벽 창이 뜨면 → **허용**을 눌러도 됩니다. (내 PC에서만 쓰는 경우가 많습니다.)

## ② 인터넷 링크로 쓰기 (조금 더 단계가 있음)

한 줄로 말하면, **프로그램 파일을 온라인 저장소에 올리고**, **Streamlit이 그걸 웹에서 돌려 주는 서비스**에 연결하면, **주소가 하나 생깁니다.**

어려운 말은 잠깐 뒤로 두고, 순서만 따라가 보세요.

### 1단계: GitHub에 코드 올리기

- **GitHub**는 인터넷에 있는 **파일 보관함** 같은 곳입니다. (회원가입이 필요합니다.)
- [github.com](https://github.com) 에 접속해서 **새 저장소(New repository)** 를 만듭니다. 이름은 마음대로 괜찮습니다.
- 저장소가 만들어지면, 이 폴더의 **파일들**을 업로드합니다.  
  **웹에서** “Add file → Upload files” 로 드래그해서 넣어도 됩니다.
- **중요:** `counseling.db` 파일은 **학생 정보가 들어 있을 수 있으니 올리지 않는 것이 좋습니다.** (이미 목록에서 빠지게 설정되어 있어요.)

### 2단계: Streamlit에 연결하기

- [share.streamlit.io](https://share.streamlit.io) 에 접속합니다.
- **GitHub 계정으로 로그인**합니다. (1단계에서 쓴 것과 같으면 편합니다.)
- **Create app** 또는 **New app** 같은 버튼을 누릅니다.
- **방금 만든 GitHub 저장소**를 선택합니다.
- **Main file path** (메인 파일 경로)에 **`app.py`** 라고 정확히 적습니다.
- **Deploy** (배포)를 누릅니다. 몇 분 기다리면 끝납니다.

### 3단계: 주소 확인하기

- 준비가 끝나면 **`https://~~~~.streamlit.app`** 형태의 주소가 나옵니다.  
  그 주소가 **선생님 앱의 전용 링크**입니다. 브라우저에 붙여 넣으면 됩니다.

### (참고) 클라우드에서 데이터가 사라질 수 있어요

인터넷으로 돌릴 때, 서버 안의 파일은 **다시 배포하거나 서버가 바뀌면 초기화**될 수 있습니다.  
**처음에는 그냥 써 보셔도 되고**, 나중에 “데이터를 계속 남기고 싶다”면 그때 **영구 저장소**나 **환경 변수 `CLASSMANAGER_DB_PATH`** 를 이야기하면 됩니다. 지금은 몰라도 괜찮습니다.
